
# Kmila Backend

Backend for version 2 of the **Kmila** application, developed in Django.  
Provides the logic and endpoints for data management, authentication, security, and services behind the client interface.

## Main features

- Modular structure based on Django applications within `apps/kmila`
- Support for deployment with Docker (Dockerfile and production configuration)
- Use of Gunicorn as a WSGI server
- Validations, security, and base configuration ready for production

## Technologies

- Python / Django
- Gunicorn
- Docker
- Mypy (static typing)
